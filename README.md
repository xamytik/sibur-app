## ✨ Возможности

| Функция | Описание |
|---|---|
| 📸 **Калибровка по фото** | Загрузи скриншот с дрона и настрой HSV-маску для своего типа покрытия |
| 🎥 **Live анализ видео** | Покадровый анализ видеопотока с HUD-оверлеем в реальном времени |
| 📐 **Расчёт площади** | Автоматический пересчёт пикселей в квадратные метры с учётом FOV и высоты |
| 💰 **Финансовый контроль** | Сравнение заявленных и реальных объёмов работ подрядчиков |

## 🚀 Быстрый старт

### 1️⃣ Установи Python
Скачай с [python.org](https://python.org/downloads) и при установке **обязательно** поставь галочку:
> ✅ Add Python to PATH

### 2️⃣ Клонируй репозиторий
```bash
git clone https://github.com/xamytik/sibur-app.git
cd sibur-app
```

### 3️⃣ Установи зависимости
```bash
pip install -r requirements.txt
```

### 4️⃣ Запусти приложение
```bash
streamlit run app.py
```

🌐 Браузер откроется автоматически: **http://localhost:8501**

## 📦 Зависимости

```
streamlit      — веб-интерфейс
opencv-python  — обработка видео и изображений
numpy          — работа с массивами пикселей
matplotlib     — визуализация данных
```

## 🎛️ Режимы мониторинга

- 🟫 **Стройка / Земля** — детекция коричневого грунта
- ⬜ **Уборка снега** — детекция очищенного асфальта
- 🟩 **Покос травы** — детекция зелёных зон
- 🎨 **Ручной режим** — полная настройка HSV вручную
